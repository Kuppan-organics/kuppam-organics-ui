# Kuppam Organics Website Theme

## Overall Design Philosophy
**Organic & Traditional** - A warm, earthy, nature-inspired theme that reflects the organic farming heritage and traditional values of Kuppam Organics.

---

## Color Palette

### Primary Colors
- **Primary (Organic Brown)**: `#6B3F1D` (hsl(24, 56%, 27%))
  - Main brand color, used for headings and key elements
  - Represents the rich, fertile soil

- **Secondary (Earth Clay)**: `#9C6B3D` (hsl(28, 44%, 42%))
  - Supporting color for secondary elements
  - Earthy, grounded tone

- **Accent (Olive Green)**: `#6E7F3A` (hsl(73, 37%, 36%))
  - Used for organic badges, highlights, and nature elements
  - Represents growth and sustainability

- **Gold (Harvest Gold)**: `#C89B3C` (hsl(40, 54%, 51%))
  - Call-to-action buttons, highlights, and premium elements
  - Represents harvest and prosperity

### Background Colors
- **Background (Warm Parchment)**: `#F7F3EE` (hsl(36, 33%, 95%))
  - Main page background
  - Soft, warm, paper-like texture

- **Card (Soft Cream)**: `#FFF9F2` (hsl(35, 100%, 97%))
  - Card and container backgrounds
  - Slightly warmer than main background

### Text Colors
- **Foreground (Dark Soil)**: `#2E1A0F` (hsl(24, 53%, 12%))
  - Primary text color
  - Deep, rich brown for excellent readability

- **Muted Foreground**: `#6B5A4A` (hsl(29, 19%, 35%))
  - Secondary text, descriptions
  - Softer brown for less emphasis

---

## Typography

### Font Families
- **Headings**: Poppins (300, 400, 500, 600, 700, 800)
  - Modern, geometric sans-serif
  - Clean and impactful for titles

- **Body Text**: Inter (300, 400, 500, 600, 700)
  - Highly readable sans-serif
  - Optimized for UI and body content

### Font Usage
- Headings use `font-heading` class (Poppins)
- Body text uses `font-body` class (Inter)
- Font weights range from 300 (light) to 800 (extra bold)

---

## Design Elements

### Border Radius
- **Base Radius**: `0.75rem` (12px)
- **XL**: `1rem` (16px)
- **2XL**: `1.25rem` (20px)
- **3XL**: `1.5rem` (24px)

### Shadows
- **Soft Shadow**: Subtle elevation for cards
- **Card Shadow**: Medium depth for interactive elements
- **Elevated Shadow**: Strong depth for hover states

### Gradients
- **Hero Gradient**: Primary brown to secondary clay
- **Gold Gradient**: Harvest gold variations
- **Organic Gradient**: Background transitions

---

## Component Styling

### Buttons
- **Primary**: Gold background with dark text
- **Secondary**: Outline style with primary color
- **Ghost**: Transparent with hover effects

### Cards
- Rounded corners (2xl)
- Soft shadows
- Cream/white backgrounds
- Hover elevation effects

### Navigation
- Fixed header with backdrop blur
- Transparent on hero sections
- Solid background on content pages
- Glassmorphic mobile menu

---

## Visual Style

### Aesthetic
- **Warm & Inviting**: Earth tones create a welcoming feel
- **Natural & Organic**: Colors inspired by nature and farming
- **Traditional with Modern Touch**: Classic values with contemporary design
- **Clean & Minimal**: Focus on content and readability

### Key Visual Elements
- Organic badges and labels
- Grain texture overlays (subtle)
- Rounded, soft shapes
- Generous white space
- Natural imagery (farm landscapes, produce)

---

## Dark Mode Support
The theme includes a dark mode variant with:
- Darker backgrounds (deep browns)
- Adjusted contrast for readability
- Maintained color relationships
- Gold accents for highlights

---

## Brand Identity
The theme reflects:
- **Heritage**: Traditional farming methods
- **Quality**: Premium organic products
- **Sustainability**: Environmental consciousness
- **Community**: Support for local farmers
- **Health**: Natural, chemical-free produce

---

## Usage Guidelines

### When to Use Each Color
- **Primary Brown**: Headings, important text, brand elements
- **Gold**: CTAs, highlights, premium features, buttons
- **Olive Green**: Organic badges, nature elements, success states
- **Earth Clay**: Secondary actions, supporting elements

### Spacing
- Consistent padding and margins
- Generous spacing for readability
- Responsive spacing for mobile devices

### Animations
- Smooth transitions (0.3s - 0.6s)
- Subtle hover effects
- Fade and scale animations
- Parallax effects on hero sections

---

## Accessibility
- High contrast ratios for text
- Clear focus states
- Readable font sizes
- Color-blind friendly palette
- Keyboard navigation support
